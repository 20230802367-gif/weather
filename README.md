## Weather App

A responsive and user-friendly Weather App that provides real-time weather information and forecasts for cities around the world. The application uses a weather API to fetch live weather data and displays temperature, humidity, wind speed, weather conditions, and other essential climate information.

## Features
Real-Time Weather Updates
Search Weather by City Name
Current Temperature Display
Humidity Information
Wind Speed Details
Weather Condition Icons
Responsive Design
Fast API Integration
Location-Based Weather Information
User-Friendly Interface

## How It Works
Enter a city name in the search box.
The application sends a request to the weather API.
Live weather data is retrieved.
Weather details are displayed instantly.
Users can search multiple cities for updated forecasts.
